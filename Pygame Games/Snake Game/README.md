# 🐍 Snake Game

A recreation of the classic Snake Game using Python and Pygame.

## ✨ Features
- Snake grows when eating food
- Game over when hitting walls or itself
- Score tracking
- Smooth controls (WASD / Arrow keys)

## 🛠️ Tech Stack
- Python 3
- Pygame

## 🚀 How to Run
1. Install pygame:
   ```bash
   pip install pygame
